# SMPS
A totally free and open source SMPS that may be dangerous

## **THIS HAS NOT BEEN TESTED! USE AT YOUR OWN RISK!**

## Specifications
- **230V 50/60hz** Input
- **0-50A** Output @1.5kW
- **5-72V** Output
- Dual **Synchronous Rectification**
- **ETD59** Transformer Core
- Upgrade Path for a **PFC**
- **12V 1A Auxilliary PSU** with Fan header and direct out
