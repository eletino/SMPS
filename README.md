# SMPS
A totally free and open source SMPS that may be dangerous

## **THIS HAS NOT BEEN TESTED! USE AT YOUR OWN RISK!**

## Specifications
- **230V 50/60hz** Input
- **0-50A** Output @1.5kW (May do 3kW continuous, limited by primary rectification)
- **5-72V** Output
- Dual **Synchronous Rectification**
- **ETD59** Transformer Core
- Upgrade Path for a **PFC**
- **12V 1A Auxilliary PSU** with Fan header and direct out
- **CC/CV** Mode
- Frequency control
- Automatic fan speed control
  - Hard switching Power pin using a MOSFET and PWM
  - Dedicated PWM pin on fan

#### Made with EASYEDA

 I have added a lot of experimental boards. The main ones are: **AUX PSU**; **Board1_alt_mosfets_NCF_TR**; **LV**; **IN**; **SWITCH**;
